# 速查表

## 一、景别黄金比例

| 景别 | 占比 | 功能 | 短剧用法 |
|------|------|------|---------|
| 特写 | 40% | 情绪放大器 | 眼、唇、手、喉结、攥拳，每段必用 |
| 近景 | 35% | 信息密度最高 | 胸以上，主打对话、对峙、暧昧 |
| 中景 | 20% | 叙事主力 | 腰/膝以上，交代互动、站位 |
| 全景 | 5% | 定调+留白 | 只用来开篇定环境或结尾留白 |

**坚决少用**：大远景、长全景——节奏慢、信息弱。

## 二、运镜五剑客

| 运镜 | 使用率 | 效果 | 提示词 |
|------|--------|------|--------|
| 固定 | 最高 | 稳定、干净 | "固定" |
| 慢推/微推 | 灵魂 | 逼近、升温 | "特写慢推" |
| 跟拍 | 移动必用 | 动态、紧迫 | "近景跟拍" |
| 正反打 | 双人标配 | 节奏快、情绪强 | "A近景→B近景→A特写" |
| 甩镜/快切 | 爆点专用 | 震惊、顿悟 | "快速切换" |

**坚决少用**：长旋转、复杂环绕、高速变焦、长轨迹运动。

## 三、10秒CLIP标准结构

```
镜1（0-3s）｜钩子：中景/全景 固定 → 定环境+人物站位
镜2（3-6s）｜发展：中景→近景 慢推 → 压情绪+对话推进
镜3（6-9s）｜转折：近景→特写 慢推/快切 → 情绪爆点
镜4（8-12s）｜卡点：特写 固定/定格 → 锁钩子+留白
```

## 六、句式轮换（必须轮换）

## 九、阶段节奏要求

| 阶段 | 钩子密度 | 反转要求 | 典型时长 |
|------|---------|---------|---------|
| 起（1-3集）| 每15-20秒 | 不强求，结尾必须有 | 15-20秒/单元 |
| 承（4-10集）| 每10-15秒 | 每集至少1次小反转 | 10-15秒/单元 |
| 转（10-70集）| **每5-10秒** | **必须反转** | **5-10秒/单元** |
| 合（70-结局）| 每10秒 | 终极反转+情感落地 | 10秒/单元 |

## 十、钩子强度

```
LV1（软）：信息预告 → "明天见"
LV2（中）：关系疑云 → "他为什么记得？"
LV3（强）：身份反转 → "原来我只是他的项目？"
LV4（爆）：生死/失控 → "她转身时，他拿起了电话"
```

**要求**：
- 常规CLIP结尾：LV2以上
- 关键转折CLIP：LV3-LV4
- 付费卡点CLIP：必须是LV4
