# 风格卡索引（Layer0）

> 只管“这条短剧整体是什么风格”，不管文戏武戏，也不管 PROMOTE 写法。

## 运行时风格卡
- `restrained_bargain`：克制博弈
- `high_density_hook`：高密度卡点爽剧
- `suspense_thriller`：悬疑推进
- `identity_reveal`：身份反转
- `revenge_comeback`：打脸反杀
- `late_rescue`：迟到援军
- `emotion_extreme`：情感极致
- `mistaken_love`：误会虐心
- `office_romance`：职场虐恋
- `comedy_flip`：喜剧反差

## 选择原则
- 看观众预期，不看镜头
- 看冲突来源：身份 / 权力 / 误会 / 证据 / 人性
- 可用主风格 + 辅风格

## 示例
- `restrained_bargain + suspense_thriller`
- `office_romance + emotion_extreme`
