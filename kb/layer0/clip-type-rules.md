# CLIP类型规则（Layer0）

> 只管“这条 CLIP 的内容执行方式”，不等于短剧风格。

## 类型
- `dialogue`：对话 / 拉扯 / 暧昧 / 对峙
- `action`：打斗 / 追逐 / 推搡 / 摔打 / 快速位移
- `suspense`：线索 / 偷听 / 遮挡 / 反光 / 来电 / 证据推进
- `comedy`：反差 / 误会 / 尴尬 / 梗点 / 节奏包袱
- `reveal`：身份揭露 / 证据公布 / 真相落地

## 判断规则
- 有明显肢体冲突、追逐、位移 → `action`
- 主要靠眼神、台词、停顿推进 → `dialogue`
- 主要靠线索、遮挡、证据、偷看推进 → `suspense`
- 主要靠反差、误会、梗、尴尬 → `comedy`
- 主要靠“真相揭开” → `reveal`
